1. 자료의 특성을 숫자 또는 문자로 표현할 수 있을 때, 이를 변수라고 한다
-> 변수는 성질에 따라 양적인 변수와 질적인 변수로 구분된다
-> 양적인 변수는 연속변수와 이산변수로 세분된다

2. 경험적 연구에서는 연구자가 피험자들을 처리집단과 통제집단으로
   배정하지 않는다
-> 피험자 중 일부는 스스로 처리 조건을 만족시킴으로써 처리집단을 이루고
   다른 피험자들은 통제집단을 이룬다
-> 예를 들어, 흡연에 관한 연구에서 흡연자들은 처리집단을 형성하고, 비흡연자들은 통제집단을 형성한다

3. 경험적 연구에서 한 요인이 다른 요인과 밀접한 관게, 즉 관련성을 갖는다는 사실은
   두 요인 간에 인과관계가 존재할 가능성을 시사한다.
-> 그러나 관련되어 있다는 사실 자체가 곧바로 인과관계의 존재를 의미하지는 않는다

4. 경험적 연구에서 처리효과는 다른 혼동요인에 의한 효과와 구별하기 어렵다
-> 이로 인해 경험적 연구로부터 잘못된 인과관계를 끌어낼 수 있다
-> 두 요인이 관련되어 있는 경우, 두 요인간 인과관계를 논하기 전에
   두 요인 모두와 곤련된 제3의 요인, 즉 혼동요인이 존재하지는 않는지 조사해 보아야 한다

5. 경험적 연구에서는 반드시 다음 질문을 해 보아야 한다
-> 피험자들이 어떻게 처리집단과 통제집단에 배정되었는가?
-> 처리여부를 제외하면 두 집단은 서로 동질적인가?
-> 어떤 요인이 처리효과를 혼동시키는가?
-> 혼동요인을 통제하기 위해 어떤 조치가 취해졌는가?
-> 그 조치는 타당한가?

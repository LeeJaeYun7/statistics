

```
통계학은 자료를 분석하는 학문이다
-> 왜 자료를 분석할까?
-> 이론만 갖고는 충분치 않기 때문이다
-> 왜 일부의 자료만을 분석할까?
-> 전체에 대한 자료가 없기 때문이다

- 물리적, 경제적 또는 시간적 제약으로 인해 일일이 전체를 조사하지 못한다
-> 따라서 일부만을 조사하여 이로부터 전체의 특성에 대해 추측해 보고자 한다
-> 이 때, 관심의 대상이 되는 전체를 모집단이라 하고, 조사된 일부를 표본이라 부른다.

-모집단의 특성을 모수라 하고, 이에 대응하는 표본의 특성은 통계량이라 한다
-> 표본을 분석하여 모집단의 특성을 알아내기 위해서는 추론의 과정이 필요하다.

- 통계학은 표본의 자료를 수집, 정리, 요약할 뿐만 아니라 요약된 자료를 토대로
  그 자료의 모태가 되는 모집단에 대해 짐작, 추측해 보는 작업을 포함한다
-> 이 때, 제대로 된 자료를 제대로 분석해야 한다
```
